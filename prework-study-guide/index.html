CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three ways to use css inline, internal style sheet,external style sheet
  • If you create and external style sheet you need to make sure that you link it to the html file or the styling wont take effect
  • * is used to apply a rule or change to all the elements on the page
  • You can referance a color not only by its name but also by the hexadecimal code which stares with a #

Git

  • git status: checks what branch we are currently on

  • git pull origin main: to pull down any changes from the base branch
